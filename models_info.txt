## Sample Model Data

- **User**
  - `id`: UUID (primary key)
  - `name`: string
  - `email`: string (unique)
  - `role`: enum (admin, branch-admin, location-admin, customer)
  - `createdAt`: timestamp
  - `updatedAt`: timestamp
  - *optional*: `profileImageUrl`: string (URL to avatar)

- **Location**
  - `id`: UUID
  - `name`: string
  - `address`: string
  - `coordinates`: { `lat`: float, `lng`: float }
  - `createdAt`: timestamp
  - `updatedAt`: timestamp
  - *optional*: `description`: string

- **Booking**
  - `id`: UUID
  - `userId`: UUID (foreign key to User)
  - `locationId`: UUID (foreign key to Location)
  - `date`: date
  - `timeSlot`: string
  - `status`: enum (pending, confirmed, cancelled)
  - `createdAt`: timestamp
  - `updatedAt`: timestamp
  - *optional*: `notes`: string

- **MenuItem**
  - `id`: UUID
  - `locationId`: UUID (FK)
  - `name`: string
  - `price`: decimal
  - `category`: string
  - `available`: boolean
  - `createdAt`: timestamp
  - `updatedAt`: timestamp
  - *optional*: `imageUrl`: string

- **Attendance**
  - `id`: UUID
  - `userId`: UUID (foreign key to User)
  - `locationId`: UUID (foreign key to Location)
  - `date`: string (YYYY-MM-DD)
  - `status`: enum (present, absent, half-day)
  - `markedBy`: UUID (User who marked attendance)
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Category**
  - `id`: UUID
  - `name`: string
  - `description`: string
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Coupon**
  - `id`: UUID
  - `code`: string (unique)
  - `discount`: decimal (percentage or amount)
  - `validFrom`: date
  - `validTo`: date
  - `usageLimit`: integer
  - `usedCount`: integer
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Expense**
  - `id`: UUID
  - `locationId`: UUID (FK)
  - `amount`: decimal
  - `type`: enum (utility, salary, maintenance, other)
  - `date`: date
  - `description`: string
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Notification**
  - `id`: UUID
  - `userId`: UUID (FK)
  - `title`: string
  - `message`: string
  - `read`: boolean
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Recipe**
  - `id`: UUID
  - `name`: string
  - `ingredients`: array of strings
  - `instructions`: text
  - `createdAt`: timestamp
  - `updatedAt`: timestamp

- **Table**
  - `id`: UUID
  - `locationId`: UUID (FK)
  - `number`: integer
  - `seats`: integer
  - `status`: enum (available, occupied, reserved)
  - `createdAt`: timestamp
  - `updatedAt`: timestamp
